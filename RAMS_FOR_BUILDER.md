# Dieter Rams — 10 Principles, applied to building Aida

> For Agent BUILDER. Each principle is a build directive, not a poster. The 10th rules them all: **as little design as possible.** When in doubt, cut.

| # | Principle | What it means for Aida (build directive) |
|---|-----------|------------------------------------------|
| 1 | **Innovative** | The **emotional note + bank account** is invented here, powered by Opus 4.8. Borrow no emotion taxonomy, no sentiment library, no copied wheel. The functional emotions *are* the product. |
| 2 | **Useful** | Every element serves the **read** or the **catch**. If a feature doesn't help Dima understand others or be understood, it doesn't ship. |
| 3 | **Aesthetic** | Calm, quiet, safe — never clinical. The surface should lower cortisol, not raise it. Soft, ambient, human. |
| 4 | **Understandable** | The read explains itself: tentative, relational language (*"reads as…", "for them this is unusual"*). No scores, no jargon, no `NEGATIVE 0.82`. |
| 5 | **Unobtrusive** | **Silent by default.** Ambient marker for incoming; the **one deliberate pulse** is the *only* alarm. It's a layer on the conversation — **never a dashboard.** |
| 6 | **Honest** | Never fake certainty. **Cold-start humility** ("still learning how they write"). Show confidence. Never manipulate the user's trust in themselves. |
| 7 | **Long-lasting** | The bank account **compounds** over time; memory is never silently dropped or truncated. Build for the relationship that deepens, not a one-off trick. |
| 8 | **Thorough, down to the last detail** | The details *are* the product: the **safety-gate re-open**, repair-by-conversation (4th-wall), the microcopy of a warning, the moment the pulse fires. Sweat them. |
| 9 | **Environmentally friendly** *(resource-honest)* | No speculative token burn. Compute on demand (no warming, no prefetch). Lean calls. Respect the phone's battery and the credit budget. |
| 10 | **As little design as possible** | Ship only the **irreducible core**: per-person read · the wound-catch · the emotional bank account. Cut everything else. **Less, but better.** |

## The one test (Dima's inhabited bar)
Before shipping any choice, ask: *"Would I want this if I were the one who couldn't feel emotion?"* If it adds friction, noise, or false certainty — remove it.

## Hard "do not" (from the principles)
- ❌ No dashboard as the surface (violates #5, and it's a banned category).
- ❌ No imported emotion taxonomy / sentiment lib (violates #1 — build fresh).
- ❌ No confident guess on a cold start (violates #6).
- ❌ No feature that doesn't serve the read or the catch (violates #2, #10).
