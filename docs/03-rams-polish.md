# Tier 3 — Dieter Rams Polish

> *Less, but better.* Cut the solution until nothing is left to remove. Each capability earns its place or dies. Default to kill.

## The cut

- **Per-person model (theory-of-mind)** — **KEEP.** The engine. Non-negotiable; it's what makes breadth-through-depth possible and unifies both problems (see `.claude/reasoning_ledger.md`).
- **Read incoming: subtext + tone overlay** — **KEEP.** The core act — what replaces threat-default with a grounded read.
- **Check outgoing: warning pulse + reframe** — **KEEP.** Where real harm is prevented. The second face of the same engine.
- **Read outgoing neutral/positive (*"Joy: …"*)** — **MERGE** into the outgoing check. Same pass; surface prominently only on a warning. No separate affirmation feature.
- **Auto-memory, segment-deletable, never injected** — **KEEP.** Integrity rule, not a feature to trim.
- **Perspective-divergence ("they think X, you think Y")** — **KEEP, but DEFER the full version.** MVP holds each person's baseline + your history. The richer self-vs-other divergence model is the second pass, once baselines are trusted.
- **Somatic Practicing** — **DEFER to secondary**, as scoped. Not in the irreducible core; a distinct mode reached deliberately.
- **Chrome extension** — **DEFER (Phase 2).** Phase 1 is the mobile web app.

## The irreducible core

A **per-person model** that learns how someone communicates and your history with them, feeding a **single read** of each message:
- **incoming** → show subtext + tone, grounded in *them*;
- **outgoing** → pulse + reframe when it would wound;

with **memory built automatically** and never hand-injected.

Remove any one and the vision breaks: drop the model → generic sentiment that can't see subtext; drop the read → no product; drop the outgoing catch → you prevent no wounds. **That is the line.**

## Deferred (with triggers)
- Somatic Practicing → after the outward read is trusted.
- Perspective-divergence ToM → after baselines are trusted.
- Chrome extension → Phase 2.
- Group / spoken → after 1:1 written is nailed.
